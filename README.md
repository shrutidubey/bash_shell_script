# bash_shell_script
Inside this Repository using shell scripting a project has been made. This project will be of help if you have multiple folders and inside those folders 
there are several files and you need to find the files which contains a particular word.

In this project. File 'ourscript2.sh' contains the source code for execution.

You can execute it by going to the path where 'shell' folder is kept in command line and then using the command ./ourscript2.sh 

Details about the project:

There are few files and folders in the folder 'shell'. Some files inside this have word sports written inside them. The script in ourscript2.sh finds the file having
word sports and puts these files inside a folder named hot-folder. Also the number of line where the word sports is found is displayed at the end of the file 
inside hot-folder.
